# StockVeda

Open source Indian stock market research platform — screening, charting, backtesting, and macro data in one self-hosted tool.

## Stack

- **Backend:** FastAPI + DuckDB + pandas-ta
- **Frontend:** React 19 + Vite + TailwindCSS + lightweight-charts

## Quickstart

```bash
# Backend
uv sync
cd backend && uvicorn main:app --reload

# Frontend
cd frontend && npm install && npm run dev
```
